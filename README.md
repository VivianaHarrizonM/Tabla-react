# 📋 Users Table

This project is an application developed with **React + Vite** that displays a table of users obtained from a public API. It allows **filtering by name** and **paging** of results.

## 🚀 Technologies used

- React
- Vite
- CSS (custom styles)
- Fetch API (to fetch data)
